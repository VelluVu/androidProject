# Android Project

> Introduction to Android programming — exercises and small applications built with Java and Android SDK.

![Language](https://img.shields.io/badge/Language-Java-red) ![Platform](https://img.shields.io/badge/Platform-Android-green?logo=android) ![Type](https://img.shields.io/badge/Type-School%20Exercises-orange)

## About

A collection of Android application exercises completed as part of a mobile programming course. The project covers the fundamentals of building Android apps using Java and the Android SDK — from UI design and activity lifecycle through to data persistence and device APIs.

## Topics Covered

- Android Activity and Fragment lifecycle
- XML layouts and UI components (TextView, Button, RecyclerView)
- Intents and navigation between screens
- SharedPreferences and local data storage
- Permissions and device APIs (camera, sensors)
- Basic networking (HTTP requests, JSON parsing)
- Event handling and listeners

## Getting Started

### Prerequisites

- Android Studio (latest stable)
- Android SDK (API level 21+)
- A physical Android device or emulator

### Running the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/VelluVu/androidProject.git
   ```
2. Open the project in **Android Studio**.
3. Sync Gradle and wait for dependencies to download.
4. Run on an emulator or connected device with **Run ▶**.

## Context

Completed as a school course module on mobile development. Android knowledge is increasingly relevant for XR and spatial computing, particularly as mixed reality platforms expand their mobile footprint.
